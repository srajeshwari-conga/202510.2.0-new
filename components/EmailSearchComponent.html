Description

The Email search Component allows users to search, select, and manage email addresses using a dynamic dropdown with checkboxes. It supports multi-selection and a "Select All" feature, providing a smooth user experience for handling email lists.

import { EmailLookupModule } from '@congarevenuecloud/elements';
@NgModule({
imports: [EmailLookupModule, ...]
})
export class AppModule {}
