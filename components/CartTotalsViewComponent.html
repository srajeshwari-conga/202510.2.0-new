Description

CartTotalsView component is used to show the detailed breakup of cart totals on the cart in a modal window.

Usage

import { CartTotalsViewModule } from '@congarevenuecloud/elements';

@NgModule({
imports: [ CartTotalsViewModule, ...]
})
export class AppModule {}

Methods

enableButton
enableButton(summaryGroup: Array<SummaryGroup>)

The method is responsible to enable the UPDATE button when the user updates the summary group field values of the cart. Checks if the updated values are equal to the initial data or if any value updated has a null AdjustmentType. By default the UPDATE button is disabled.
