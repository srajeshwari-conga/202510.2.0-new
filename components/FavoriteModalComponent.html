Description

The Favorite modal allows users to add products or cart line items from the drawer to a Favorite Configuration. The modal provides options to select from existing favorites or create a new favorite, enabling users to add or update items within the favorite configuration.

Preview

Usage

import { FavoriteModalModule } from '@congarevenuecloud/elements';
@NgModule({
imports: [FavoriteModalModule, ...]
})
export class AppModule {}
