Description

The ReplaceProductModalComponent is used to display a modal for replacing products based on replacement constraint rules. To use this component, the rules input property should be provided with the relevant constraint rule groups. The modal supports two different behaviors based on the ShowCheckboxInRuleDialog flag within the configSelectConfigProductssettings configuration setting:

  • Multiple product selection using checkboxes: When ShowCheckboxInRuleDialog is set to true, checkboxes appear next to each product, allowing users to select and replace multiple products at once.
  • Single product replacement using buttons: When ShowCheckboxInRuleDialog is set to false, checkboxes are hidden, and each product displays its own Replace button. In this mode, only one product can be replaced at a time.
  • Refer to the Conga Revenue Reference Documentation for more information.

@Component({
  ...
})
export class AppComponent {

  replaceProductModal: ReplaceProductModalComponent;

  openReplaceModal() {
    this.replaceProductModal.openModal();
  }
}
