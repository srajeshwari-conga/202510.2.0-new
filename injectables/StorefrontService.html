getPollingConfig()

This method fetches the configuration data related to the maximum number of retries and delay duration for pricing circuit from the ConfigSystemProperties for flow associated to Storefront and returns an Observable emitting an object containing these values.

Example:

  ngOnInit() {

      this.maxRetries = data.maxRetries;
      this.delay = data.delay;

  }

getSettingsByKey
getSettingsByKey(key: SettingsName)

Fetches the settings based on the given key.

This method retrieves specific settings that control various aspects of the configuration. The key helps identify which setting needs to be fetched.

Example Usage:

  ngOnInit() {
    this.setting$ = this.storefrontService.getSettingsByKey(SettingsName.SOME_SETTING);
    this.setting$.subscribe(setting => {
      this.specificField = setting['desiredField'];
    });
  }
}

getStorefronts
getStorefronts(filters: Array<FieldFilter>, page: number, limit: number, sortField?: string)

Retrieves a list of storefront records based on the provided filters. This method allows you to search for storefronts using various field filters and supports pagination and sorting.

Example:

import { StorefrontService, Storefront } from '@congarevenuecloud/ecommerce';
import { FieldFilter, FilterOperator } from '@congarevenuecloud/core';
import { Observable } from 'rxjs';

export class MyComponent implements OnInit {
  storefronts$: Observable<Array<Storefront>>;

  constructor(private storefrontService: StorefrontService) {}

  ngOnInit() {
    // Filter by store URL
    const filters: Array<FieldFilter> = [
      {
        field: 'StoreUrl',
        value: 'https://example.com',
        filterOperator: FilterOperator.EQUAL
      }
    ];

    this.storefronts$ = this.storefrontService.getStorefronts(filters, 1, 10, 'Name');
    this.storefronts$.subscribe(storefronts => {
      console.log('Found storefronts:', storefronts);
    });
  }
}
Parameters :
Name Type Optional Default value Description
filters Array<FieldFilter> No

List of field filters to apply to the storefront search
page number No 1

The page number to retrieve. Defaults to 1.
limit number No 10

The number of storefront records to include in the result set. Defaults to 10.
sortField string Yes

The field by which to sort the storefront records (optional)
Returns : Observable<Array<Storefront>>

An observable containing an array of Storefront records matching the filters
