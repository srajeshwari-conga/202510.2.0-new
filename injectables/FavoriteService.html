getMyFavorites
getMyFavorites(searchString?: string, filters?: Array<FieldFilter>, limit: number, pageNumber: number, orderBy?: string, orderDirection?: "ASC" | "DESC")

This method fetches the list of favorites for user's active pricelist in the application.

By default, it returns all favorites associated to the pricelist, that are either public or created by the user.

Example:

import { Favorite, FavoriteResult, FavoriteService } from '@congarevenuecloud/ecommerce';
import { Observable } from 'rxjs/Observable';
export class MyComponent implements OnInit {
result: FavoriteResult;
result$: Observable<FavoriteResult>;
constructor(private favoriteService: FavoriteService){}

getMyFavorites(): Observable<FavoriteResult> {
this.favoriteService.getMyFavorites(searchString, [{ field: 'Id', value: null, filterOperator: FilterOperator.NOT_EQUAL }], 20, 1, 'ModifiedDate', 'DESC').subscribe(r => this.result = r);
// or
results$ = this.favoriteService.getMyFavorites(searchString, [{ field: 'Id', value: null, filterOperator: FilterOperator.NOT_EQUAL }], 20, 1, 'ModifiedDate', 'DESC');
}
}
Parameters :
Name Type Optional Default value Description
searchString string Yes

string to search the favorites with.
filters Array<FieldFilter> Yes

list of favorite field filters to filter the favorite records.
limit number No 10

number representing the maximum number of favorite records to be fetched.
pageNumber number No 1

number representing the current pagination page.
orderBy string Yes

string representing the sort field on the favorite.
orderDirection "ASC" | "DESC" Yes

string representing the sort direction. Posible values are 'ASC' or 'DESC'.
Returns : Observable<FavoriteResult>

An observable containing list of favorites matching the criteria with total favorites count.
