Description

Add comments component provides a flexible form for adding comments/notes to business objects. The component supports both modal and inline layouts with configurable options for comment types and validation.

Usage

import { CommentsModule } from '@congarevenuecloud/elements';

@NgModule({
imports: [CommentsModule, ...]
})
export class AppModule {}

Modal Layout

Inline Layout

// Basic usage - Modal layout
```typescript
<apt-add-comments
  [record]="quote"
  [layout]="'modal'"
  [title]="'Add Comment'"
  (onAction)="handleCommentAction($event)">
</apt-add-comments>
// Inline layout with configuration
```typescript
<apt-add-comments
  [record]="proposal"
  [layout]="'inline'"
  [config]="{
    showCommentType: true,
    submitButtonText: 'Submit',
    cancelButtonText: 'Cancel'
  }"
  (onAction)="handleCommentAction($event)">
</apt-add-comments>

Configuration options for the comment form including modal options, button text, and behavior settings. See AddCommentsConfig for more details on the structure and fields required

Type : string

Custom CSS class to apply to the component for additional styling.

Type : "modal" | "inline"
Default value : 'modal'

The layout mode for the comment form. 'modal' opens in a modal dialog, 'inline' embeds in the page.
record
Type : AObject

The business object record to associate the comment with.
title
Type : string

The title text displayed at the top of the comment form.
