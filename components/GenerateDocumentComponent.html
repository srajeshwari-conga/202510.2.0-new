Description

The Generate-document component is used to generate the document for a custom Order or Quote.

Usage

import { ButtonModule } from '@congarevenuecloud/elements';

@NgModule({
imports: [ButtonModule, ...]
})
export class AppModule {}
// Basic Usage
```typescript
<apt-generate-document
             [record]="order"
></apt-generate-document>
```html
// With Custom Button Classes
```typescript
<apt-generate-document
             [record]="order"
             buttonClasses="btn btn-success px-3 custom-generate-btn"
></apt-generate-document>
